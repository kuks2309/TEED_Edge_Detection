# Claude 작업 지침

## 핵심 원칙
1. **사용자가 지시한 사항만 수행한다**
2. **임의로 기능을 추가하거나 변경하지 않는다**
3. **작업 완료 후 지시한 내용만 했는지 확인하고 보고한다**
4. **객과적인 사실을 가지고 판단한다**

## 현재 프로젝트 상태
- **모델**: CNN 단일 패턴 인식 (EdgeCNN)
- **학습 방식**: Positive 데이터만 사용 (label=1로 학습)
- **Negative 데이터**: 사용하지 않음
- **추론**: CNN 출력 confidence (0~1) - 높으면 학습된 패턴과 유사

## 금지 사항
- Negative 데이터 수집/학습 로직 추가 금지
- Autoencoder로 변경 금지
- 사용자가 요청하지 않은 모델 구조 변경 금지
- 요청하지 않는 코딩 금지
- 코딩 전에 프로그램 구조 먼저 제시하고 코딩은 승인 받을 것

## 코드 수정 규칙
- 기존에 지시한 코드 로직을 임의로 변경하지 않는다
- 이전에 승인받은 코드 구조를 유지한다
- 변경이 필요하면 반드시 사유를 설명하고 승인받는다

### 작업 종료 전
- [ ] 사용자가 요청한 작업만 수행했는가?
- [ ] 요청하지 않은 기능을 추가하지 않았는가?
- [ ] 기존 로직을 임의로 변경하지 않았는가?
- [ ] 매 답변마다 다짐과 확인을 보고했는가?
